# tools
一个工具库

- `getType` 返回参数的类型
- `deepClone` 深拷贝  *待优化*
- `isEmpty` 判断是否为空
- `toUnique` 去重
- `ajax` 原生 `ajax` 请求，`promise`封装
- `debounce` 函数防抖
- `throttle` 函数节流
- `getRem` 获取当前`1rem`对应的`px`值
- `checkNum` 校验字符串是否为数值
- `loadImageAsync` 懒加载图片
- `getBrowser` 浏览器判断
- `buildScheme` 生成浏览器跳转`APP`的`scheme`
- `generateIntent` 生成`H5`跳转`APP`链接
- `dateFormat` 返回对应格式的日期时间
- `getParentsNode` 原生实现获取指定父级元素
